# VoguePoint - интернет-магазин одежды

Этот проект представляет собой сайт магазина женской одежды "VoguePoint", состоящий из клиентского приложения для пользователей и библиотеки UI-компонентов. Проект разделен на две основные части:

**frontend/** - Клиентское приложение для пользователей (главная страница, каталог, авторизация, контакты)
**ui-library/** - Библиотека переиспользуемых UI-компонентов

## Технологии
- React 19 с TypeScript
- Vite для сборки и dev-сервера
- Jest + Testing Library для тестирования
- ESLint для линтинга
- CSS Modules для стилизации
- React Router DOM v6 для маршрутизации
- Google Fonts для типографики

## Компоненты библиотеки

### 1. Card (Карточка товара)
  - Карточка товара для каталога:
  - Заголовок "вещь гардероба"
  - Место для фото товара
  - Описание товара (текст)
  - Интерактивная стрелка для перехода в каталог
  - Адаптивный дизайн
  - Эффекты при наведении

### 2. Input (Форма авторизации)
  - Форма для регистрации/авторизации пользователей:
  - Поля: Имя, Email, Телефон, Пароль
  - Чекбокс согласия на обработку данных
  - Валидация полей
  - Анимация при появлении
  - Адаптивный дизайн

### 3. AboutLink (Кнопка "О нас")
  - Интеллектуальная кнопка навигации:
  - При клике со страницы "О нас" плавно скроллит к соответствующей секции
  - При клике с других страниц ведет на главную к секции "О нас"
  - Стилизована под навигационное меню
  - Поддержка кастомных параметров

### 4. Contacts (Контакты)
  - Страница контактной информации:
  - Заголовок "КОНТАКТНЫЕ ДАННЫЕ"
  - Разделы: телефон, почта, адрес
  - Адаптивный дизайн
  - Единообразное оформление

### 5. ProductCard (Карточка продукта) - в разработке
  - Детальная карточка продукта:
  - Основное изображение товара
  - Варианты цветов/оттенков
  - Выбор размера
  - Кнопка добавления в избранное
  - Кнопка добавления в корзину
  - Информационные табы (описание, уход)

### 6. Basket (Корзина) - в разработке
  - Компонент корзины покупок:
  - Список выбранных товаров
  - Управление количеством
  - Итоговая сумма
  - Кнопка оформления заказа

## Инструкция по запуску

### Предварительные требования
- Node.js 18+
- npm или yarn

### Установка и запуск

1. Клонирование репозитория
git clone <repository-url>
cd vogue_point

2. Установка зависимостей для ui-library
cd ui-library
npm install
npm run build

3. Установка зависимостей для frontend
cd ../frontend
npm install

3. Запуск приложения
npm run dev

4. Открытие в браузере
Приложение будет доступно по адресу: http://localhost:5173

Скрипты проекта
UI Library (ui-library/)
npm run build - сборка библиотеки

npm run dev - разработка с вотчером

npm run test - запуск тестов

npm run lint - проверка кодстайла

npm run test:coverage - запуск тестов с генерацией отчета о покрытии кода

Frontend (frontend/)
npm run dev - запуск dev-сервера

npm run build - сборка для production

npm run test - запуск тестов

npm run lint - проверка кодстайла

Тестирование

Тестирование ui-library
cd ui-library
npm run test
npm run test:coverage

Тестирование frontend
cd ../frontend
npm run test
npm run test:coverage
